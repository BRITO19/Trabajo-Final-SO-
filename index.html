Debe escoger un sistema operativo de la actualidad e investigar los siguientes tópicos:


1. Características de windows 10

  • Conexión entre varios dispositivos

    • Windows 10 puede ser instalado en distintos equipos con el fin de que vivas la misma experiencia de este sistema operativo en celulares, tabletas o cualquier otro dispositivo compatible.
    Según el dispositivo que tengas, Windows 10 transformará su diseño para que puedas usarlo fácilmente.

  • Ventana de Inicio

    • En Windows 10 regresa el tradicional menú de Inicio que desapareció en la versión anterior. Aquí se combina la pantalla de Inicio con las placas de Windows 8 y el tradicional menú de
    Inicio de Windows 7 y sistemas operativos anteriores a este.

  • Aplicaciones para todos los dispositivos

    • Esta versión del sistema operativo de Microsoft cuenta con algunas aplicaciones diseñadas para que se adapten a cualquier dispositivo: computadores, celulares, tabletas o Xbox One.
    Estas aplicaciones se adaptan a cada equipo y le muestran al usuario una interfaz fácil de manejar.

  • Centro de notificaciones

    • Como en los dispositivos móviles, Windows 10 trae un centro de notificaciones y acciones en la parte derecha de la pantalla desde donde podrás activar o desactivar herramientas como el wifi,
    el VPN (tecnología para conectar uno o más computadores a una red privada), el modo avión, el Bluetooth.

  • Cortana

    • Windows 10 trae una característica muy importante: Cortana, su asistente virtual, el cual controlas a través de comandos de voz. Puedes pedirle información sobre el clima, sobre tus vuelos,
    hacer búsquedas en internet o dentro de tu equipo, entre otras cosas.

  • Microsoft Edge

    • Este navegador llegó a reemplazar a Internet Explorer. Trae algunas novedades como filtros de seguridad para proteger descargas, se pueden escribir textos en las páginas de internet e intenta
    predecir qué página vas a abrir después de la que estás visitando para que su carga sea más rápida.

  • Integración con Xbox

    • Algunas de las funciones de Xbox fueron integradas con Windows 10 para que desde cualquier dispositivo puedas acceder a ella. Podrás comentar y ver comentar los logros de tus amigos.

  • Windows Hello

    • Esta herramienta presente en Windows 10 te ayuda a que inicies sesión de una forma rápida y ágil con solo ver o tocar el equipo. Esta función permite la detección de rostros y huellas digitales
    para acceder al dispositivo. Para ejecutar esta herramienta tu computador necesita un lector de huellas y una cámara compatible con los estándares de Windows Hello.

2. Ventajas y desventajas

    Ventajas de Windows 10

  • Interfaz de Usuario Amigable: Combina lo mejor de Windows 7 y Windows 8, ofreciendo un menú de inicio mejorado, fácil de usar y personalizable.

  • Compatibilidad con Aplicaciones: Ofrece una amplia compatibilidad con la mayoría del software del mercado, incluyendo aplicaciones antiguas y modernas.

  • Actualizaciones Regulares: Actualizaciones Regulares: Proporciona actualizaciones de seguridad y de características de forma regular, mejorando continuamente la seguridad y la funcionalidad del sistema.

  • Seguridad Mejorada: Incluye Windows Defender, que ha mejorado significativamente, y otras características de seguridad como Windows Hello para una autenticación más segura.

  • Modo Tablet y Compatibilidad con Pantallas Táctiles: Ofrece una experiencia de usuario mejorada en dispositivos híbridos y tablets, con interfaces optimizadas para el tacto.

  • Cortana: Integra este asistente digital, permitiendo control por voz y facilitando la realización de tareas cotidianas.

  • Microsoft Edge: Presenta un navegador web más rápido y seguro que su predecesor, Internet Explorer, con mejores características de privacidad y rendimiento.

    Desventajas de Windows 10

  • Problemas de Privacidad: Ha habido preocupaciones sobre la cantidad de datos personales y de uso que Microsoft recopila por defecto.

  • Actualizaciones Forzadas: Las actualizaciones automáticas pueden ser intrusivas, a menudo reiniciando el sistema en momentos inoportunos y a veces introduciendo nuevos problemas.

  • Compatibilidad de Hardware: Algunos dispositivos antiguos pueden no ser compatibles o pueden experimentar problemas de rendimiento con Windows 10

  • Rendimiento en Equipos Antiguos: Aunque tiene buen soporte para hardware antiguo, en algunos casos puede ser más lento que versiones

  • Publicidad y Bloatware: Incluye anuncios y aplicaciones preinstaladas que no siempre son útiles para el usuario y pueden consumir recursos

  • Interfaz Cambiante: La constante evolución del sistema puede confundir a los usuarios, con cambios frecuentes en la interfaz y en la ubicación de ciertas funciones.

  • Costo: Aunque ofrece actualizaciones gratuitas desde versiones anteriores bajo ciertas condiciones, la adquisición de una nueva licencia de Windows 10 puede ser costosa para algunos usuarios o empresas.

3. Estabilidad y rendimiento

Microsoft fue lanzando numerosas actualizaciones que mejorando notablemente Windows 10, tanto a nivel de interfaz como de funciones, de seguridad y de estabilidad, y al final dieron forma a uno de los
mejores sistemas operativos que ha lanzado la compañía hasta el momento, y se mantiene como una de las mejores opciones a día de hoy.

Windows 10 es más estable, está más madurado, presenta menos errores y lleva una racha más razonable en todo lo que respecta a los fallos derivados de actualizaciones. Por otro lado, también es un sistema operativo
que se lleva bien con hardware antiguo, y es capaz de funcionar de forma óptima con configuraciones que no son compatibles con Windows 11.

La interfaz de Windows 10 es bastante equilibrada, ya que reúne lo mejor del Windows más clásico con algunos de los aciertos de Windows 8.1.

4. Nivel de seguridad

Windows 10 incluyen Seguridad de Windows, que proporciona la protección antivirus más reciente. El dispositivo se protegerá activamente desde el momento en que inicies Windows. Seguridad de Windows examina
continuamente en busca de malware (software malintencionado), virus y amenazas de seguridad. Además de esta protección en tiempo real, las actualizaciones se descargan automáticamente para ayudar a mantener
tu dispositivo seguro y protegerlo de amenazas.

Windows Defender ayuda a estar protegido del malware que puede entrar a través de Internet, memorias USB o al copiar archivos desde una unidad de red al ordenador.

Por otro lado, Windows Defender también permite controlar el cortafuego (firewall) de Windows para vigilar todas las conexiones entrantes y salientes del sistema. No obstante, presenta un contra y es que es bastante
complicado de configurar si se carece de conocimientos técnicos.

Otras funcionalidades de protección que integra son las que permiten controlar la memoria de las aplicaciones y del sistema y protegerlo de vulnerabilidades gracias a su sistema anti-exploits. Además, se puede ver
el estado general del PC para detectar problemas de rendimiento o controlar las cuentas que hacen uso del ordenador.

Sin embargo, este antivirus presenta dos desventajas más. Por un lado, este endpoint ralentiza considerablemente el ordenador y por otro, no detecta el 100% de las amenazas.

5. Costo de licencia de Windows 10

El precio de una licencia de Windows 10 y 11 varía dependiendo de la edición. Si bien es cierto que los precios de las distintas versiones de Windows han ido descendiendo en el tiempo desde Windows XP, mucha gente
y empresas siguen prefiriendo hacerse con diversas versiones del sistema operativo de los de Redmond por algunas vías algo más económicas.

Los precios de las licencias de Windows dependen del lugar donde las compremos. Podemos arrancar con los precios oficiales de la web de Microsoft e ir descendiendo hasta los precios que ofrecen otra serie de tiendas
en internet, dependiendo, eso sí, de la versión y edición que compremos.

En cuanto a las versiones de Windows 10, nos movemos en una horquilla desde los 145 euros para la versión Home, pasando por los 259 euros para Windows 10 Pro, hasta los 439 euros de la versión Enterprise para empresas.
